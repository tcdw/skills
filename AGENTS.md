# Repository Guidance

This repository stores personal Agent Skills.

## Structure

- Put reusable skills under `skills/<skill-name>/SKILL.md`.
- Keep `SKILL.md` frontmatter `name` equal to the folder name.
- Use `docs/` for repository-level guidance and references.

## Quality Bar

- Descriptions must say when to use the skill, not just what it is.
- Skills should be operational and concise.
- Do not commit secrets, API keys, local tokens, or generated private config.
